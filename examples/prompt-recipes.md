# Prompt Recipes

Use these as starting points for demos, screenshots, and real work.

## 1. SaaS dashboard

```text
Create a polished desktop analytics dashboard for a B2B SaaS product with a sidebar, KPI cards, trend charts, recent activity, and a strong hierarchy for decision-making.
```

## 2. Pricing page

```text
Create a pricing page for an AI developer tool with 3 plans, a comparison table, trust indicators, and a clear upgrade path for teams.
```

## 3. Admin panel

```text
Design an internal admin panel for operations teams with table-heavy workflows, quick filters, status chips, detail drawers, and compact density.
```

## 4. E-commerce mobile checkout

```text
Create a mobile-first checkout flow for a premium e-commerce brand with a clean cart summary, address step, payment step, and reassuring trust cues.
```

## 5. CRM workspace

```text
Design a desktop CRM workspace with lead stages, account summaries, timeline events, notes, and task follow-ups optimized for sales reps.
```

## 6. Data-heavy BI homepage

```text
Create a business intelligence homepage for execs with sharp summary cards, narrative insight panels, chart blocks, and one obvious next action.
```

## 7. Team settings page

```text
Design a settings page for a multi-user SaaS product with tabs for members, roles, billing, usage, and API access using an enterprise-clean visual style.
```

## 8. AI tool onboarding

```text
Create an onboarding flow for an AI design tool that explains the product quickly, asks for goals, and makes the first successful action feel immediate.
```

## 9. Marketplace listing page

```text
Design a plugin marketplace page with search, categories, featured plugins, installation cards, and lightweight editorial curation.
```

## 10. Variants prompt

```text
Explore three different visual directions for this screen. Vary layout, typography, density, and color emphasis while keeping the same product structure.
```
