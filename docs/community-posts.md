- prompt -> useful UI direction

## FAQ snippets for replies

### What makes it different?

```text
The raw SDK is flexible, but StitchFlow focuses on the workflow layer: installability, prompts, reusable skill packaging, local artifacts, and cross-client support.
```

### Why local HTML and screenshots?

```text
Because teams usually need artifacts they can review and build from immediately, not only a hosted generation experience.
```

### Who should try it first?

```text
Product/design engineers and agent builders are the best first-fit users.
```
